# 블루아카이브 싯딤 OnDeviceAI

## 프로젝트 개요
갤럭시 탭 9에서 동작하는 자율 AI 캐릭터.  
- 음성 명령 지원
- 3D 모델링 및 실시간 애니메이션
- 상자 제어 및 사용자 지원

## 준비사항
1. Android 개발 (Kotlin / Java)
2. On-Device AI (TensorFlow Lite, PyTorch Mobile)
3. 음성 인식(STT) 및 합성(TTS)
4. 3D 모델링 & 렌더링 (Blender, Unity3D)
5. 데이터 관리 & 로컬 상태
6. AI 행동 설계 (FSM, Behavior Tree)
7. 성능 최적화 (GPU/NPU, 양자화, 배터리 관리)
